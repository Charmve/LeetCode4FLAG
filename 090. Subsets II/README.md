## Description

Given a collection of integers that might contain duplicates, ``nums``, return all possible subsets (the power set).

<strong>Note</strong>: The solution set must not contain duplicate subsets.

<strong>Example:</strong>
```
Input: [1,2,2]
Output:
[
  [2],
  [1],
  [1,2,2],
  [2,2],
  [1,2],
  []
]
```
